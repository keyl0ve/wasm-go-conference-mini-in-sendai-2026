# wasm-go-conference-mini-in-sendai-2026

[Go Conference mini in Sendai 2026](https://sendaigo.jp/)（2026年2月21日開催）の登壇資料・デモコードです。

## 内容

### great-mqrio-bros

Ebitengine + WebAssembly で作ったマリオ風ゲームのデモです。

### lgtm-generator

Go + WebAssembly で作った LGTM 画像ジェネレーターです。

### session-demo

セッション中に使用するデモコード集です。

| ディレクトリ  | 内容                                   |
| ------------- | -------------------------------------- |
| `add`         | 基本的な Go WebAssembly の例           |
| `async`       | 非同期処理のデモ                       |
| `debug`       | デバッグ手法のデモ                     |
| `stacktrace`  | スタックトレースの確認                 |
| `panic`       | panic のハンドリング（良い例・悪い例） |
| `memory-leak` | メモリリークのパターンと対策           |
